# 产品体验上下文

本上下文定义 LanJing 的跨媒体发现、媒体消费与资料库语义。它消费规则系统提供的标准媒体资源，不记录组件、布局、实现细节或临时方案。

## Language

**境场（Realm）**：
跨媒体发现上下文。它组织不同来源与媒体类型的标准媒体资源，帮助用户发现内容并进入对应媒体空间。
_Avoid_：把境场称为某一种媒体的首页

**媒体空间（Media Space）**：
围绕一种媒体类型建立的一等产品上下文，例如小说空间、音乐空间或影视空间。它拥有符合该类型的发现、组织、详情与消费语义，并可与境场互相进入。
_Avoid_：用“媒体应用”指代整个类型体验

**前台活动（Foreground Activity）**：
用户当前投入主要注意力的浏览或消费活动。任一时刻只有一个前台活动，它决定当前媒体空间与主要交互表面。
_Avoid_：主任务、当前页面

**环境音频会话（Ambient Audio Session）**：
可在境场与不同媒体空间之间持续的一条音频消费会话。它能与不占用音频焦点的前台活动并存，但在其他活动请求音频焦点时让出播放。
_Avoid_：后台播放器、第二前台活动

**规则工作室（Rule Studio）**：
应用内用于管理来源并创建、检查、调试规则的高级工具上下文。它与内容消费上下文分离，但可接收内容界面发起的来源诊断或规则定位请求。
_Avoid_：来源设置页、开发者工具页

**应用设置（App Settings）**：
用户调整 **本地产品偏好** 的专用上下文（例如外观：主题模式、外观包、材质透明度）。它不是四境导航之一，也不承担媒体发现或消费；与规则工作室分离，不替代来源/规则配置。
_Avoid_：把设置做成第五媒体空间；与规则工作室/来源管理混为同一入口

**来源范围（Source Scope）**：
用户对聚合浏览结果施加的显式来源过滤条件。未指定来源范围时，境场与媒体空间默认展示所有可用来源的资源。
_Avoid_：当前来源、全局源切换

**备选线路（Alternative Route）**：
标准媒体模型明确关联到同一消费目标的其他来源或资产路径。只有存在明确关系时，内容界面才允许切换备选线路。
_Avoid_：根据标题猜测的自动换源

**资料库（Library）**：
用户跨媒体持有的收藏、历史、消费进度、离线资源与自建集合。资料库由用户整体拥有，不按媒体空间复制。
_Avoid_：把书架、歌单或片单当作彼此独立的资料库

**资料库投影（Library Projection）**：
媒体空间按自身语义呈现资料库内容的类型专属视图，例如小说书架或音乐歌单。投影不拥有独立副本，对它的操作直接更新共享资料库。
_Avoid_：媒体资料库、副资料库

**活书架（Living Shelf）**：
小说空间对书籍资料库的默认投影。它在同一书架中按当前阅读状态与最近活动调整书籍权重，并直接呈现阅读进度。
_Avoid_：把继续阅读与书架拆成重复区域

## 边界

- 产品体验消费规则系统提供的标准媒体资源、关系、动作和展示提示。
- 前端模板负责发现、详情、阅读、播放、收藏和管理体验；媒体类型差异通过标准模型和模板语义表达。
- 来源规则不能定义 UI 布局、组件或来源专属页面结构。
- 只有标准媒体模型明确关联的来源或资产路径，才能作为备选线路展示。
- 内容界面可以发起来源诊断或规则定位请求，但不直接依赖规则内部流程节点。
- 资料库由用户整体拥有，媒体空间只提供类型专属投影，不创建副本。
